# Use Claude Code in Cursor

You don’t need a separate “Claude Code” app. Install the extension from inside Cursor.

---

## Step 1: Install the extension in Cursor

1. In Cursor, press **Cmd + Shift + X** (Mac) or **Ctrl + Shift + X** (Windows) to open **Extensions**.
2. In the search box, type **Claude Code** (or **Claude**).
3. Find the one by **Anthropic** (e.g. “Claude Code” or “Claude”) and click **Install**.
4. If it asks you to reload, click **Reload** (or restart Cursor).

---

## Step 2: Sign in / add API key

When you use the extension for the first time it will ask you to:

- Sign in with your **Anthropic** account, or  
- Add an **API key** from [console.anthropic.com](https://console.anthropic.com).

You need one of: API key (pay-as-you-go), Claude Pro/Max, or a Team/Enterprise plan.

---

## Step 3: Open Claude Code in Cursor

The extension often **doesn’t show a sidebar icon** in Cursor. Open it like this:

1. Press **Cmd + Shift + P** (Mac) or **Ctrl + Shift + P** (Windows) to open the **Command Palette**.
2. Type **Claude Code** (or **Open in Side Bar**).
3. Run **“Claude Code: Open in Side Bar”** (or similar).
4. The Claude panel should open (often on the right). You can start a chat there.

**Optional:** To give it a shortcut: **Cmd + K** then **Cmd + S** → search for **“Claude Code: Open in Side Bar”** → assign a key (e.g. **Cmd + Shift + C**).
